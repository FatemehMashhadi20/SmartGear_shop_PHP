# SmartGear-Shop

Welcome to **SmartGear Shop**, a simple eCommerce website where users can browse and purchase various electronic gadgets. This project includes a homepage, product detail page, and basic styling to give users a visually appealing and interactive shopping experience.

## Features

- **Homepage**: Displays a list of products with names, prices, descriptions, and a call-to-action button to view the products.
- **Product Detail Page**: Allows users to view detailed information about a product, including an image, description, price, quantity selection, and an "Add to Cart" button.
- **Responsive Design**: The site adjusts to different screen sizes, making it mobile-friendly.
- **Search Bar**: Provides a search functionality to quickly find products (currently without actual search implementation).
- **Cart Icon**: A cart icon at the top allows users to view their cart (no actual cart functionality yet).

## Installation

To run this project locally, follow these steps:

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/SmartGear-Shop.git 
   
2. Navigate to the project directory:

```bash cd SmartGear-Shop ```

3. Ensure you have PHP installed on your system. You can check by running:

```bash php -v ```

4. Run the PHP built-in server to view the site:
   
```bash php -S localhost:8080 ```

## CSS Styling
The project uses a simple yet elegant CSS layout:

**Flexbox Layout**: For arranging the product image and product info side by side on large screens, and stacking them vertically on smaller screens.

**Hover Effects**: The product images have a zoom effect when hovered over, adding a dynamic feel to the site.

**Responsive Design**: The site adjusts its layout for different screen sizes using media queries.
